BookShop is a full-stack web application that allows users to upload, manage, and read digital books online.
The platform supports two user roles: writers (who can submit books for admin approval) and admins (who review, approve, or delete books). 
All book covers and PDFs are securely stored in Cloudinary, with automatic file cleanup on deletion. 
The frontend is built with React and Tailwind CSS, providing a modern, responsive user experience, 
while the backend uses Node.js, Express, and MongoDB for robust data management and authentication.

How to Use BookShop

Admins:
- Log in to access the admin dashboard.
- Review and approve or reject new book submissions.
- Approve or reject deletion requests from writers.
- All approved books are visible to users for reading and download

Writers:
- Register and log in to your account.
- Submit new books by uploading a cover image and PDF file.
- Your submissions will be sent to the admin for approval.
- You can request deletion of your own books, which will also require admin approval.

Users/Readers:
- Anybody can read/download from it and its totally free for everyone.
